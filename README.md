# Snake-Game
A custom built Snake Game (snake eats apples and grows). This is a culmuilative project I made while taking CS3520 at Northeastern.

# How to setup:
First, ensure you're on a linux server with a clang compiler.

Then, call:
``` c++
make

./snake_game
```
voila
